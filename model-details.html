Model Details
DeepFIPM forecasts post-peak solar flare irradiance using a fully connected neural network (FCN) trained on 60 minutes of pre-peak GOES soft X-ray (0.1–0.8 nm) observations, targeting a 180-minute post-peak irradiance sequence at 1-minute cadence.
DeepFIPM은 태양 플레어 발생 직전 60분간의 GOES 연X선(0.1–0.8 nm) 관측 데이터를 입력으로 받아, 플레어 정점 이후 180분간의 복사량 변화를 1분 간격으로 예측하는 완전연결신경망(FCN) 기반 모델입니다.
FCN MODEL — SOFT X-RAY BAND (0.1–0.8 nm)
The final pre-peak observation is used as an anchor; the network predicts a 180-minute residual correction added to that anchor to produce the forecast.
마지막 관측값을 기준점(anchor)으로 삼아, 신경망은 180분간의 보정값(잔차)을 예측하고 이를 기준값에 더해 최종 예측 복사량을 산출합니다.
Input
60-min pre-peak GOES X-ray
↓
Anchor
Final pre-peak value concatenated to input
↓
Hidden Layers [230 → 922 → 230]
Depth 2 · GELU · no BatchNorm · Dropout 0.257
↓
Residual Δ
180-min correction sequence
↓
Output
Anchor + Δ = 180-min irradiance
Direct fixed-window regression (no autoregression); LR 4.8e-04, batch size 16. Log10-transformed irradiance in/out.

Prediction Framework
Each flare's 60-minute pre-peak X-ray input maps to a 180-minute post-peak irradiance target across four wavelength bands, using FISM2 reconstructed irradiance as the reference.
각 플레어의 정점 직전 60분 X선 입력을 이용해, 정점 이후 180분간 4개 파장 대역(연X선 + EUV 3개)의 복사량을 FISM2 참조값 기준으로 예측합니다.
Reference
Kim, J.-H. et al., (2026) "Benchmarking Deep Learning Models for Real-Time Prediction of Post-Peak Solar Flare Irradiance Using FISM2 Data." Submitted to Nature Scientific Reports.
